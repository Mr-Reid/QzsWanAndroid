# QzsWanAndroid
基于 wanandroid.com 开发的 MVP + Retrofit2 + RxJava2 +okhttp3 开发的 Android APP

### 目前已完成的功能（持续更新）

- 登录注册登出
- 首页文章列表
- 首页搜索文章
- 项目分类
- 项目列表
- 收藏文章
- 取消收藏
- 收藏列表

### dependenices
